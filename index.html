常見問題解答

快速找到問題解決方案
➕

如何新增人員資料？

▼
1. 進入「卹滿照護」功能模組
2. 點擊「新增人員」按鈕
3. 填寫完整的人員資訊表單
4. 上傳照片（可選）
5. 點擊「儲存」完成新增
📊

如何匯出資料報表？

▼
1. 在卹滿照護頁面選擇年份和月份
2. 選擇匯出格式：CSV（Excel格式）或PDF（報告格式）
3. 點擊對應的匯出按鈕
4. 系統會自動下載或開啟檔案
📷

如何檢查照片完整性？

▼
1. 選擇要檢查的年份和月份
2. 點擊「檢查照片」按鈕
3. 系統會顯示照片完整性統計
4. 查看哪些人員缺少照片
✅

如何標記案件完成？

▼
1. 在人員資料卡片中找到狀態區域
2. 點擊「完成」按鈕
3. 系統會自動記錄完成時間
4. 如需取消，可點擊「設為未完成」
💾

如何備份和還原資料？

▼
1. 進入「系統設定」功能
2. 在資料管理區域找到備份按鈕
3. 點擊「備份資料」下載備份檔案
4. 還原時點擊「還原資料」並選擇備份檔案
🌗

如何切換主題模式？

▼
1. 可在頂部導航欄點擊主題按鈕
2. 或進入「系統設定」→「外觀設定」
3. 點擊「切換主題」按鈕
4. 系統會在淺色和深色主題間切換
📞
